Superglobals

$GLOBALS References all variables available in global scope
$_SERVER An array containing information such as headers, paths, and script locations.
$GET this contains an associative array of variables passed to the current script using query parameters in the URL
$POST this contains an associative array of variables passed to the current script using a form submitted using the “POST” method
$_FILES An associative array of items uploaded to the current script via the HTTP POST method. The structure of this array is outlined in the POST method uploads section.
$_COOKIE An associative array of variables passed to the current script via HTTP Cookies.
$_SESSION An associative array containing session variables available to the current script.
$_REQUEST his contains the contents of $_GET, $_POST, and $_COOKIE
$_ENV An associative array of variables passed to the current script via the environment method.
